# Netflix User Insight Dashboard

## Overview
This Power BI project presents a comprehensive analysis of **Netflix user behavior, revenue trends, device usage, and geographic distribution**.

The dashboard was developed during my **Business Analytics Internship at InLighnX Global Pvt. Ltd. (Jun 2025 â€“ Jul 2025)** and is shared here for **portfolio and demonstration purposes**.

The objective of this project is to convert raw user data into meaningful insights that support **business strategy, customer segmentation, and subscription optimization**.

---

## Project Objectives
- Analyze monthly revenue generated from Netflix subscriptions  
- Understand user growth patterns across quarters  
- Identify device preferences among Netflix users  
- Examine geographic distribution of users  
- Compare revenue contribution by gender and subscription type  

---

## Key Insights
- **Premium subscription plans** contribute the highest monthly revenue  
- User growth peaks during **Q3**, followed by a slight decline in Q4  
- **Smart TVs and smartphones** are the most commonly used devices  
- Revenue contribution is **nearly balanced across genders**  
- Netflix users are globally distributed, with strong presence in **North America and Europe**  

---

## Dashboard Features
- KPI cards for **Total Revenue, Total Users, and Average User Age**  
- Quarterly trend analysis of user growth  
- Revenue breakdown by **subscription type**  
- Device usage distribution across users  
- Geographic visualization of users by country  
- Interactive slicers for **Device Type** and **Subscription Type**
